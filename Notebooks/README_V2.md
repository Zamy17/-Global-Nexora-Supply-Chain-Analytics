# Global Nexora Supply Chain Analytics Platform

Global Nexora Supply Chain Analytics Platform is an end-to-end analytics project that demonstrates how modern data engineering and business intelligence can be applied to optimize supply chain operations.

The project covers the complete analytics lifecycle, starting from raw data ingestion, data profiling, cleaning, feature engineering, exploratory data analysis, and data validation, followed by the implementation of a dimensional data warehouse using a Star Schema architecture in MySQL.

The analytical layer is powered by SQL views and optimized indexes, enabling interactive dashboards and business reporting with Tableau.

This project was designed to simulate a real enterprise data analytics environment and showcase practical skills required for Data Analyst, Business Intelligence Analyst, Supply Chain Analyst, and Data Engineer roles.
